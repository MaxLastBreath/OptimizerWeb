PC Setup Guide

  • Download NX-Optimizer.
  • Run the Executable.
    • IMPORTANT!!! Do not run the executable from the Downloads Folder.
  • At the top right Ensure the text matches your Emulator "Legacy" or "Ryujinx".
  • Change the settings as you see fit.
  • If you are running your Emulator in "Portable/User" mode. Press the Browse Button and select your Emulator's Executable.
  • Pressing Apply will now install the mod(s) to your Emulator.

    • Switch Setup Guide

  • Download NX-Optimizer.
    • IMPORTANT!!! Do not run the executable from the Downloads Folder.
  • Run the Executable.
  • Change the settings as you see fit. Preferably select the "Switch" Preset.
  • Press Extract.
    • This will now generate a local folder with the mod's name besides NX-Optimizer.exe.
  • Copy the Content of the mod to your switch's sdcard.
    • Please Note you need the TitleID of the game.
    • Copy paste the content inside of the folder exefs and romfs to : sdcard:/ atmosphere/ content/TitleID/.
    • This should result in 2 folders named exefs and romfs.

    Manual Setup Guide PC.

  • Download NX-Optimizer.
    • IMPORTANT!!! Do not run the executable from the Downloads Folder.
  • Run the Executable.
  • Change the settings as you see fit.
    • IMPORTANT!!! Doing a manual installation will require you to manually change some settings...
    • IMPORTANT!!! For a mod that changes shadows/resolution set Upscaling to 1.0x and DRAM to 6GB or above...
  • Press Extract.
    • This will now generate a local folder with the mod's name besides NX-Optimizer.exe.
  • Copy the folder to your Emulator's Mod directory for your game.
    • You can usually access the game by right clicking and selecting BROWSE MOD DIRECTORY.
